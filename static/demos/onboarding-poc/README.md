A simple, zero-dependency onboarding checklist application.

## Features
- **No Installation Required**: Runs directly in the browser.
- **Persistent Progress**: Your progress is saved automatically (using `localStorage`).
- **Gamified Experience**: Visual progress tracking and a celebration upon completion.

## How to Run
1.  Navigate to the `onboarding-poc` folder.
2.  Double-click `index.html`.
3.  The application will open in your default web browser.

## How to Deploy (GitHub Pages)
1.  Push this repository to GitHub.
2.  Go to **Settings** > **Pages**.
3.  Select `main` branch as the source.
4.  Your site will be live in minutes!
